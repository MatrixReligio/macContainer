# 현지화 용어집

MacContainer는 명령, 식별자, 경로, 이미지 참조, 신호, 단위, 코드 구문, 제품 이름을 번역하지 않습니다. 다음 용어를 사용자용 번역의 기준으로 사용합니다.

| English | 한국어 |
|---|---|
| container | 컨테이너 |
| image | 이미지 |
| runtime | 런타임 |
| volume | 볼륨 |
| registry | 레지스트리 |
| compatibility check | 호환성 검사 |
| rollback | 롤백 |
| complete uninstall | 완전 제거 |
| residue audit | 잔여 항목 감사 |
| template | 템플릿 |
| Activity Center | 활동 센터 |
| fail closed | 안전하게 차단 |
| administrator approval | 관리자 승인 |

`Apple container`, `MacContainer`, CLI 플래그, 시맨틱 버전, 해시, 아키텍처 이름, `REMOVE APPLE CONTAINER`와 같은 확인 문구는 번역하지 않습니다.
