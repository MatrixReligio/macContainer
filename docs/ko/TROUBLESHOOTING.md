<a id="troubleshooting"></a>
# 문제 해결

[영문 소스](../en/TROUBLESHOOTING.md) · 소스 리비전 `f94970774a25e899b7fb4a623d35c555d11f12e2`

오류는 안정된 코드, 원인, 삭제 처리된 진단, 재시도 안전 여부, 복구 작업을 표시합니다. 활동 센터에서 항목별 전체 기록을 확인하십시오.

<a id="first-steps"></a>
## 먼저 확인할 사항

macOS 26 이상, Apple Silicon, 정식 Release 앱인지 확인하고 개요에 런타임 상태가 보이는지 확인한 뒤 현재 영역을 한 번 새로 고칩니다. 설치, 업데이트, 롤백 또는 제거가 진행 중이면 종료 상태까지 기다립니다. 앱에서 안전하다고 명시하지 않는 한 권한 대화상자나 파괴적 작업을 반복하지 마십시오.

<a id="error-map"></a>
## 오류 및 복구 매핑

| 오류 또는 복구 ID | 의미 | 안전한 다음 조치 |
|---|---|---|
| `error.authentication` | 레지스트리 또는 권한 부여 데이터가 거부되었습니다. | 자격 증명을 편집하고 한 번 재시도. |
| `error.upstream-data` | 예상 업스트림 데이터를 디코딩할 수 없습니다. | 메타데이터를 새로 고치고 후보를 설치하지 않음. |
| `error.registry` | 레지스트리 요청이 실패했습니다. | 주소와 자격 증명을 확인하고 제공될 때만 재시도. |
| `error.helper` | 고정 권한 요청이 거부되었거나 사용할 수 없습니다. | 승인을 검토하고 활동 센터 열기. |
| `error.compatibility` | 카탈로그, 패키지, 증명 또는 검사가 실패했습니다. | 호환성 보고서를 보고 보류를 우회하지 않음. |
| `error.lifecycle` | 설치, 업데이트, 롤백 또는 제거가 검증되지 않았습니다. | 복구를 재개하고 트랜잭션 기록 따르기. |
| `error.container` 등 | 해당 리소스 작업이 실패했습니다. | 대상을 검토하고 안전 표시 시에만 재시도. |
| `uninstall.recovery.*` | 잔여 항목이 있거나 검증할 수 없습니다. | 접근 권한을 복원하고 잔여 감사 재실행. |

매개변수 검증은 필드의 안정 키와 해당 복구 키를 표시합니다. 강조된 값을 수정하십시오. 잘못된 값은 자동 전달되지 않습니다.

<a id="update-holds"></a>
## 런타임 업데이트 보류

알 수 없는 버전, 잘못된 카탈로그, 실제 기기 증명 누락, 승인되지 않은 원본, 패키지 ID 불일치, 롤백 지점 누락, 사전 또는 사후 검사 실패는 자동 설치를 중지합니다. “작업 활성”은 모든 작업이 유휴 상태가 될 때까지 기다리고 “승인 필요”는 자동 동의 또는 도우미 승인이 없음을 뜻합니다. 새 버전 검사 실패 시 증명을 차단하고 롤백합니다. **복구 필요**가 표시되면 런타임 작업을 중지하고 안내된 작업을 따르며 복구 데이터를 수동 삭제하지 마십시오.

<a id="uninstall-problems"></a>
## 제거 문제

“제거 미완료”는 하나 이상의 범주가 남아 있거나 검증할 수 없음을 뜻합니다. 리졸버와 패킷 필터에는 관리자 권한이 필요할 수 있고 Keychain 자격 증명은 사용자 컨텍스트에서 제거됩니다. 소유자 변경, 심볼릭 링크, 예상치 못한 파일 형식은 의도적으로 거부됩니다. 더 넓은 디렉터리를 삭제하지 말고 보고된 문제를 수정하십시오. 정확한 범위는 [완전 제거](COMPLETE_UNINSTALLATION.md)를 참조하십시오.

<a id="diagnostic-redaction"></a>
## 진단 삭제 처리

권한 부여 헤더, Bearer/Basic 자격 증명, 암호, 비밀 정보, API 키, 토큰, 사용자 이름, URL 내 자격 증명, 비공개 임시 경로, 홈 디렉터리 사용자 이름을 대체합니다. 실패 세부 정보는 한 줄과 제한된 길이로 저장되며 보안 필드는 평문을 영구 저장하지 않습니다. 공유 전에 고급 설정의 삭제 처리된 진단을 검토하고 Keychain 항목, 토큰, 개인 키, 컨테이너 데이터를 보내지 마십시오.

## 지원
