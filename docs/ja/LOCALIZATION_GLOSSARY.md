# ローカライズ用語集

MacContainerでは、コマンド、識別子、パス、イメージ参照、シグナル、単位、コード、製品名を翻訳しません。次の用語をユーザー向け翻訳の基準とします。

| English | 日本語 |
|---|---|
| container | コンテナ |
| image | イメージ |
| runtime | ランタイム |
| volume | ボリューム |
| registry | レジストリ |
| compatibility check | 互換性確認 |
| rollback | ロールバック |
| complete uninstall | 完全アンインストール |
| residue audit | 残留項目の監査 |
| template | テンプレート |
| Activity Center | アクティビティセンター |
| fail closed | 安全側で拒否 |
| administrator approval | 管理者の承認 |

`Apple container`、`MacContainer`、CLIフラグ、セマンティックバージョン、ハッシュ、アーキテクチャ名、`REMOVE APPLE CONTAINER`などの確認フレーズは翻訳しません。
