<a id="troubleshooting"></a>
# 故障排除

[英文源文档](../en/TROUBLESHOOTING.md) · 源版本 `f94970774a25e899b7fb4a623d35c555d11f12e2`

错误会显示稳定代码、易懂说明、已脱敏诊断、重试是否安全以及恢复操作。活动中心包含完整的逐项记录。

<a id="first-steps"></a>
## 首先检查

确认系统为 macOS 26 或更高版本、使用 Apple 芯片、应用来自规范 Release，并能在概览看到运行时状态。刷新当前域一次。如果安装、更新、回滚或卸载正在进行，请等待其到达终态。除非应用明确说明可安全重试，否则不要反复批准权限或重试破坏性操作。

<a id="error-map"></a>
## 错误与恢复映射

| 错误或恢复 ID | 含义 | 安全操作 |
|---|---|---|
| `error.authentication` | 镜像仓库或授权信息被拒绝。 | 编辑凭据后重试一次。 |
| `error.upstream-data` | 无法解码预期上游数据。 | 刷新元数据，不要安装该候选。 |
| `error.registry` | 镜像仓库请求失败。 | 检查地址与凭据；仅在提供时重试。 |
| `error.helper` | 固定特权请求被拒绝或不可用。 | 检查授权并打开活动中心。 |
| `error.compatibility` | 目录、软件包、证明或探测失败。 | 查看兼容性报告，不要绕过暂停。 |
| `error.lifecycle` | 安装、升级、回滚或卸载未验证。 | 选择继续恢复并遵循事务记录。 |
| `error.container` 等域错误 | 对应资源操作失败。 | 检查资源；仅在标记安全时重试。 |
| `uninstall.recovery.*` | 残留仍存在或无法验证。 | 恢复权限并重跑残留审计。 |

参数校验会显示字段稳定键和匹配的恢复键；修正高亮值即可，无效值不会自动分派。

<a id="update-holds"></a>
## 运行时更新暂停

未知版本、无效目录、缺少实体证明、未批准的源版本、软件包身份不符、无回滚点或前后置探测失败都会停止自动安装。“工作活跃”会等待全部运行工作空闲；“需要授权”表示缺少自动同意或帮助程序授权。新版本探测失败时会封锁其证明并回滚。显示**需要恢复**时应停止运行时工作并遵循明确操作，不要手动删除回滚数据。

<a id="uninstall-problems"></a>
## 卸载问题

“卸载未完成”表示至少一类残留仍存在或无法验证。解析器和包过滤器可能需要管理员权限，Keychain 凭据在用户上下文移除。所有权改变、符号链接或意外文件类型会被有意拒绝；请修复报告的问题，不要删除更宽目录。准确范围参见[完全卸载](COMPLETE_UNINSTALLATION.md)。

<a id="diagnostic-redaction"></a>
## 诊断脱敏

授权标头、Bearer/Basic 凭据、密码、敏感信息、API 密钥、令牌、用户名、URL 中的凭据、私有临时路径和主目录用户名会被替换；失败详情限制为单行和固定长度。安全字段不持久化明文。分享前请检查高级设置中的脱敏诊断，绝不要发送 Keychain 项、令牌、私钥或容器数据。

## 支持
