# 本地化术语表

MacContainer 的命令、标识符、路径、镜像引用、信号、单位、代码片段和产品名称保持原样。以下术语是用户界面翻译的统一标准。

| English | 简体中文 |
|---|---|
| container | 容器 |
| image | 镜像 |
| runtime | 运行时 |
| volume | 存储卷 |
| registry | 镜像仓库 |
| compatibility check | 兼容性检查 |
| rollback | 回滚 |
| complete uninstall | 完全卸载 |
| residue audit | 残留审计 |
| template | 模板 |
| Activity Center | 活动中心 |
| fail closed | 默认拒绝 |
| administrator approval | 管理员批准 |

不要翻译 `Apple container`、`MacContainer`、CLI 标志、语义版本、哈希值、架构名称或 `REMOVE APPLE CONTAINER` 等确认短语。
