# Localization glossary

MacContainer keeps commands, identifiers, paths, image references, signals, units, code spans, and product names unchanged. The terms below are authoritative for user-facing translations.

| English | Simplified Chinese | Traditional Chinese | Japanese | Korean |
|---|---|---|---|---|
| container | 容器 | 容器 | コンテナ | 컨테이너 |
| image | 镜像 | 映像檔 | イメージ | 이미지 |
| runtime | 运行时 | 執行階段 | ランタイム | 런타임 |
| volume | 存储卷 | 儲存卷宗 | ボリューム | 볼륨 |
| registry | 镜像仓库 | 映像檔登錄庫 | レジストリ | 레지스트리 |
| compatibility check | 兼容性检查 | 相容性檢查 | 互換性確認 | 호환성 검사 |
| rollback | 回滚 | 回復 | ロールバック | 롤백 |
| complete uninstall | 完全卸载 | 完整解除安裝 | 完全アンインストール | 완전 제거 |
| residue audit | 残留审计 | 殘留項目稽核 | 残留項目の監査 | 잔여 항목 감사 |
| template | 模板 | 範本 | テンプレート | 템플릿 |
| Activity Center | 活动中心 | 活動中心 | アクティビティセンター | 활동 센터 |
| fail closed | 默认拒绝 | 預設拒絕 | 安全側で拒否 | 안전하게 차단 |
| administrator approval | 管理员批准 | 管理員核准 | 管理者の承認 | 관리자 승인 |

Do not translate `Apple container`, `MacContainer`, CLI flags, semantic versions, hashes, architecture names, or confirmation phrases such as `REMOVE APPLE CONTAINER`.
