# 本地化術語表

MacContainer 的指令、識別碼、路徑、映像檔參照、訊號、單位、程式碼片段和產品名稱保持原樣。以下術語是使用者介面翻譯的統一標準。

| English | 繁體中文 |
|---|---|
| container | 容器 |
| image | 映像檔 |
| runtime | 執行階段 |
| volume | 儲存卷宗 |
| registry | 映像檔登錄庫 |
| compatibility check | 相容性檢查 |
| rollback | 回復 |
| complete uninstall | 完整解除安裝 |
| residue audit | 殘留項目稽核 |
| template | 範本 |
| Activity Center | 活動中心 |
| fail closed | 預設拒絕 |
| administrator approval | 管理員核准 |

不要翻譯 `Apple container`、`MacContainer`、CLI 旗標、語意版本、雜湊值、架構名稱或 `REMOVE APPLE CONTAINER` 等確認詞句。
