<a id="troubleshooting"></a>
# 疑難排解

[英文來源文件](../en/TROUBLESHOOTING.md) · 來源版本 `f94970774a25e899b7fb4a623d35c555d11f12e2`

錯誤會顯示穩定程式碼、易懂說明、已遮蔽診斷、重試是否安全，以及復原操作。活動中心包含完整的逐項記錄。

<a id="first-steps"></a>
## 先進行這些檢查

確認系統為 macOS 26 或更新版本、使用 Apple 晶片、應用程式來自標準 Release，並能在概覽看到執行階段狀態。重新整理目前領域一次。如果安裝、更新、回復或解除安裝正在進行，請等候其到達終態。除非應用程式明確表示可安全重試，否則不要反覆核准權限或重試破壞性操作。

<a id="error-map"></a>
## 錯誤與復原對照

| 錯誤或復原 ID | 意義 | 安全操作 |
|---|---|---|
| `error.authentication` | 映像檔登錄庫或授權資料遭拒。 | 編輯憑證後重試一次。 |
| `error.upstream-data` | 無法解碼預期的上游資料。 | 重新整理中繼資料，不要安裝候選項目。 |
| `error.registry` | 映像檔登錄庫要求失敗。 | 檢查位址與憑證；只有在提供時才重試。 |
| `error.helper` | 固定特殊權限要求遭拒或無法使用。 | 檢查授權並開啟活動中心。 |
| `error.compatibility` | 目錄、套件、證明或探測失敗。 | 查看相容性報告，不要略過暫停。 |
| `error.lifecycle` | 安裝、升級、回復或解除安裝未驗證。 | 選擇繼續復原並依照交易記錄。 |
| `error.container` 等領域錯誤 | 對應資源操作失敗。 | 檢查資源；僅在標示安全時重試。 |
| `uninstall.recovery.*` | 殘留項目仍存在或無法驗證。 | 恢復權限並重新執行稽核。 |

參數驗證會顯示欄位穩定鍵和相符的復原鍵；修正醒目提示的值即可，無效值不會自動分派。

<a id="update-holds"></a>
## 執行階段更新暫停

未知版本、無效目錄、缺少實機證明、未核准來源版本、套件身分不符、無回復點或前後置探測失敗，都會停止自動安裝。「工作進行中」會等候所有執行工作閒置；「需要授權」表示缺少自動同意或輔助程式授權。新版本探測失敗時會封鎖其證明並回復。顯示**需要復原**時應停止執行階段工作並依照明確操作，不要手動刪除回復資料。

<a id="uninstall-problems"></a>
## 解除安裝問題

「解除安裝未完成」表示至少一類殘留仍存在或無法驗證。解析器與封包過濾器可能需要管理員權限，Keychain 憑證在使用者環境移除。擁有權變更、符號連結或非預期檔案類型會被刻意拒絕；請修正回報的問題，不要刪除更廣泛的目錄。精確範圍請參閱[完整解除安裝](COMPLETE_UNINSTALLATION.md)。

<a id="diagnostic-redaction"></a>
## 診斷遮蔽

授權標頭、Bearer/Basic 憑證、密碼、機密資訊、API 金鑰、權杖、使用者名稱、URL 中的憑證、私人暫存路徑和個人專屬目錄名稱都會被替換；失敗詳細資訊限制為單行和固定長度。安全欄位不會保存純文字。分享前請檢查進階設定中的已遮蔽診斷，絕不要傳送 Keychain 項目、權杖、私密金鑰或容器資料。

<a id="support"></a>
## 支援

請提供應用程式、macOS 與執行階段版本、穩定錯誤程式碼、操作 ID、活動時間和已遮蔽診斷，不要提供憑證。查看[支援說明](../../SUPPORT.md)或聯絡 [contact@matrixreligio.com](mailto:contact@matrixreligio.com)。漏洞請依[安全性政策](../../SECURITY.md)私下回報。
