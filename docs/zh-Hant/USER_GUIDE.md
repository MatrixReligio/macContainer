# MacContainer 使用者指南

[英文來源文件](../en/USER_GUIDE.md) · 來源版本 `f94970774a25e899b7fb4a623d35c555d11f12e2`

MacContainer 以原生介面管理已審查的 Apple container 執行階段，不需要終端機。每項操作都會顯示生效值、安全性影響、進度和復原資訊。

<a id="getting-started"></a>
## 開始使用

第一次啟動時選擇**從簡易模式開始**。選擇目標，填入映像檔參照以及需要的工作區或連接埠，再選擇**檢查所有值**。檢查頁會顯示原生操作、產生值的來源及其與 Apple 預設值的差異；只有選擇**執行**後才會開始。

八種內建情境不可修改，並優先採用唯讀根檔案系統、合適的本機網路、正常停止，以及依主機產生的 CPU 與記憶體建議。共享個人專屬資料夾和巢狀虛擬化預設為關閉。需要完整合約參數時可切換到進階控制，已輸入的值不會遺失。

<a id="domains"></a>
## 功能領域

- **概覽**顯示執行階段健康狀態、相容性、執行中的工作和最安全的後續操作。
- **容器**支援建立、啟動、停止、檢查、複製、輸出、移除和互動工作階段。
- **映像檔**支援提取、推送、載入、儲存、標記、檢查和移除 OCI 映像檔。
- **建置**設定原生建置，並在活動中心顯示進度。
- **虛擬機器**管理執行階段虛擬機器、設定、日誌和預設選擇。
- **網路**管理並稽核網路、DNS 與封包過濾器狀態。
- **儲存卷宗**管理持久儲存；**映像檔登錄庫**透過**登入**和安全輸入管理憑證。空白清單表示尚未儲存經審核的憑證，請先登入再重新整理。
- **系統**顯示狀態、版本、磁碟用量、日誌、設定、服務和核心選擇。

選擇資源列可查看詳細資訊。重新整理會載入目前領域；破壞性操作會明確列出受影響資源和復原預期。

<a id="machine-workflow"></a>
## 虛擬機器工作流程

開啟**虛擬機器**並選擇**新增虛擬機器**。簡易模式會預先選擇 **Linux 虛擬機器**；檢查產生的虛擬機器名稱、映像檔、CPU 和記憶體，然後選擇**執行**。虛擬機器預設會在建立後啟動；建立時個人專屬資料夾共享和巢狀虛擬化保持關閉。

選擇一台虛擬機器後可使用**設定**修改 CPU 與記憶體，或明確啟用唯讀個人專屬資料夾共享和巢狀虛擬化。**啟動**和**停止**會作用於選取的虛擬機器；**刪除**前一律顯示受影響的虛擬機器識別碼。

<a id="parameters"></a>
## 參數與檢查

每個上游參數都有本地化標籤、簡要說明、完整用途、預設值、格式、重複與順序、相依性與衝突、平台限制、安全性或資料影響、範例、驗證訊息和復原操作。按一下欄位旁的資訊按鈕即可查看。必填欄位有標記，重複值保留輸入順序，路徑與 URL 在分派前驗證。最終檢查會標示修改值和高影響功能；正式環境操作透過型別安全橋接完成，不會產生 shell 指令。

<a id="templates"></a>
## 範本

開啟**設定 → 預設值與範本 → 開啟範本庫**，選取內建範本即可查看用途和操作；內建範本不可修改。使用**新增**或**製作複本**建立自訂範本，在詳細資料中編輯名稱、操作和工作負載欄位，然後選取**儲存**。自訂範本會持久儲存在 Application Support，之後可繼續選取、編輯、輸出或刪除。輸入會移轉支援的舊架構，並在儲存前阻擋機密資訊或未知高影響欄位；憑證與授權資料永遠不會寫入範本。

<a id="terminal"></a>
## 互動式終端機

內嵌終端機會阻擋遠端剪貼簿、連結、通知和標題變更。**中斷連線**會保留工作負載，**終止**會傳送已審查的終止訊號。啟用「減少動態效果」後，輸出不使用裝飾動畫。

<a id="activity-settings"></a>
## 活動中心與設定

活動中心記錄操作進度和每項結果。錯誤包含穩定程式碼、易懂原因、已遮蔽診斷、重試是否安全及復原操作。設定包括一般、執行階段、執行階段更新、相容性、預設值與範本、進階和關於。切換語言會等候未儲存工作及進行中的操作結束，再安全地重新啟動。

<a id="keyboard-accessibility"></a>
## 鍵盤與輔助使用

Command–N 建立情境，Command–R 重新整理，Command–Shift–L 開啟活動中心，Command–1 至 Command–9 切換概覽到系統。主要控制項都有輔助使用標籤和穩定識別碼；鍵盤焦點清楚，表格與檢查器不需滑鼠即可使用，並支援動態字體、增加對比和減少動態效果。

<a id="data-support"></a>
## 資料、輸出與支援

預設僅在本機處理，不傳送分析或遙測。只有主動執行或啟用的更新模式才會存取網路。範本輸出不是映像檔或儲存卷宗備份，請另外備份持久資料。請參閱[安裝](INSTALLATION.md)、[執行階段更新](RUNTIME_UPDATES.md)、[完整解除安裝](COMPLETE_UNINSTALLATION.md)與[疑難排解](TROUBLESHOOTING.md)，或聯絡 [contact@matrixreligio.com](mailto:contact@matrixreligio.com)。
