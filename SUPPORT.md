# Support

MacContainer is pre-release software. Before requesting help:

1. Confirm the Mac and OS satisfy the requirements in [README](README.md).
2. Reproduce with the current `main` build or newest supported release.
3. Remove credentials, tokens, usernames, registry paths, and personal data from
   diagnostics.
4. Search existing GitHub issues.

For reproducible bugs, use the GitHub bug form. For a private support question,
email contact@matrixreligio.com with the app version, macOS version, Apple
container version, expected result, observed result, and redacted steps.

Never send passwords, private keys, registry tokens, or unredacted container
content. Suspected vulnerabilities belong in the private process described by
[Security](SECURITY.md), not a support issue.
